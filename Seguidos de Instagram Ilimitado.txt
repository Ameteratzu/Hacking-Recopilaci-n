-PASO 1-

ENTRAR A LA GOOGLE PLAY Y DESCARGAR "TERMUX".
LINK DE LA APP (TERMUX): https://play.google.com/store/apps/details?id=com.termux&hl=es

-PASO 2-

ENTRAR A LA APLICACION Y PONER LOS SIGUENTES COMANDOS:

ATENCION: 
- SE PONEN EN EL ORDEN DADO, UNO DETRAS DEL OTRO.
- SI EN ALGUN MOMENTO TE APARECE UNA PREGUNTA QUE TE DA A ELEGIR ENTRE "Y/N" PONER SIEMPRE "Y"

COMANDOS: (COPIAR Y PEGAR PARA EVITAR ERRORES)

    termux-setup-storage
    pkg upgrade && update
    pkg install git
    pkg install curl
    pkg install openssh openssl openssl-tool -y
    git clone https://github.com/xd20111/inshackle
    ls

COLOCAR CADA COMANDO EN ESE ORDEN, LUEGO DE COLOCAR UNO SE PROCESARA Y TE DEJARA PONER EL OTRO Y ASI SUCESIVAMENTE CON CADA UNO.

-PASO 3-

LOS COMANDO SIRVEN PARA PODER INICIAR EL BOT QUE SUBIRA LOS SEGUIDORES, ESOS COMANDOS SOLO SE TENDRAN QUE PONER 1 VEZ, LUEGO PARA ABRIR EL BOT SOLO COLOCAR EL SIGUIENTE COMANDO:

COMANDO: (ESTE INICIA EL BOT DE SEGUIDORES).

bash inshackle.sh

-PASO 4-

UNA VEZ YA ABIERTO EL BOT, APARECERA UN MENU CON VARIAS OPCIONES, HAY QUE COLOCAR EL NUMERO "2" Y DARLE A ENTER, LUEGO LES PEDIRA "NOMBRE DE USUARIO" Y "CONTRASEÃ‘A", UNA VEZ COLOCADO ESOS DATOS YA EMPEZARA SOLO A SUBIR LOS SEGUIDORES.

DURA 1 HORA EL PROCESO ENTERO, IGUAL SE PUEDE PARAR EN CUALQUIER MOMENTO CERRANDO LA APPLICACION.

ATENCION:

- LAS SIGUIENTES VECES QUE QUIERAN VOLVER A ENTRAR AL BOT NO TIENEN QUE HACER TODOS LOS PASOS ANTERIORES, SOLO TENDRAS QUE HACER EL PASO "3" Y "4".

- SI EL "NOMBRE DE USUARIO DE INSTAGRAM" TIENE PUNTOS (.) EN SU NOMRE, ES MEJOR QUE SE CAMBIEN POR BARRAS BAJAS (_) PORQUE LOS PUNTOS (.) SUELEN BUGEARSE A LA HORA DE COLOCAR EL "NOMBRE DE USUARIO DE INSTAGRAM" EN EL BOT.
